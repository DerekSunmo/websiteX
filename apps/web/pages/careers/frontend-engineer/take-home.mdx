## 💪 挑战题项目 —— Zeam

接下来就是您大展身手的时刻到了～

我们将做一个简单的前端项目：

### 📒 项目背景

假设客户要求开发一款迷你版 Steam，名叫 **Zeam**；目的是让用户在网页中浏览感兴趣的游戏，并且可以点击查看详情页面等等。

以下就是客户给出的项目完成要求：

1. 一个首页

   - 展示所有游戏
   - 可以按照游戏的类型 与 游戏的发行时间进行排序
   - 可以快速将游戏添加到收藏夹
   - 可以快速根据关键字搜索过滤游戏
   - 可以打开收藏夹页面

2. 一个游戏详情页面

   - 展示游戏的详细信息
   - 可以快速将游戏添加到收藏夹

3. 一个收藏夹页面
   - 展示收藏的游戏
   - 可以快速将游戏从收藏夹中移除
   - 可以进入游戏的详情页面

### 🧰 API 说明

- API 获取全部游戏的接口为：[https://www.freetogame.com/api/games](https://www.freetogame.com/api/games)
- API 获取单个游戏的接口为：[https://www.freetogame.com/api/game?id=1](https://www.freetogame.com/api/game?id=1)

游戏数据 json 结构大体如下：

```json
{
  "id": 100,
  "title": "游戏名称",
  "thumbnail": "游戏缩略图",
  "short_description": "游戏简介",
  "description": "全长度游戏描述",
  "genre": "游戏类型",
  "platform": "游戏平台",
  "developer": "开发商",
  "release_date": "发行日期",
  "screenshots": [
    {
      "id": 1024,
      "image": "游戏截图地址"
    }
  ]
}
```

## 🧑🏻‍💻 项目技术要求

- 前端框架选择 React.js 加分，也可以选择使用 Vue.js （3.0+ Composition API 优先）
- 必须使用 TypeScript
- 自由发挥项目的页面设计
- CSS 框架没有限制，使用 TailwindCSS 可加分，也可以选择 Bootstrap/Ant Design 等组件库
- （可选）将网站部署到一个可访问的 URL 有加分
