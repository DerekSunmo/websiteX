## 岗位职责

- 根据业务需求文档以及设计稿完成前端研发工作，与后端紧密配合集成页面迭代等工作；
- 根据业务需求，封装高复用性、高可维护性以及高健壮性的前端组件以及负责前后端数据交互、页面交互动效以及设计系统的搭建；
- 负责公司前端的测试计划和优化，编写设计文档，沉淀和积累前端的测试框架和发布规范；
- 共同协商产品需求所需的技术支持及分析相应的需求的合理性，根据任务排期时间，自行拆分整体任务，分配合理时间；

## 任职要求

- 5 年以上相关工作经验，熟练运用但不限制于：
  - React (Hooks) 优先、Vue.js 3 (Composition API 优先) 前端框架
- TypeScript 编译
- Webpack/Babel/Vite 构建器
- TailwindCSS/PostCSS/Sass 样式系统
- Jest/Ava 测试工具
- REST/GraphQL API 交互
- SPA/PWA 应用架构
- ESLint，Prettier 配置
- Husky、Lint Staged、Git 等技术；
- 带领 Web 前端团队制定相应的工作计划；
- 熟练地将前端与后端 API 实现交互，稳定地对前端应用状态进行管理，并对前端系统性能优化深入了解；
- 热烈欢迎细节控！您对每一行代码每一次组件封装都有着细心打磨精神，连注释也不会轻易放过；
- 我们希望您对各种前端趋势有敏税的工匠触觉，富有设计创造力与激情，有探索研究精神；
- 我们希望您时刻保持好奇精神，善于协同沟通和团队协作，对于极致体验和高雅代码有追求，富有激情，对工作有较强的责任感、高执行力；
